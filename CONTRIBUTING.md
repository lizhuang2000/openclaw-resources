# Contributing to OpenClaw Resources

Thank you for your interest in contributing!

## How to Contribute

### Adding New Resources

1. Fork this repository
2. Add your resource to the appropriate section in README.md
3. Ensure the format is consistent with existing entries
4. Submit a pull request

### Resource Categories

- **Official Resources** - Links to official OpenClaw channels
- **Documentation** - Tutorials and guides
- **Plugins** - Community-built plugins and extensions
- **Videos** - Video tutorials and demos
- **Related Projects** - Projects that work with OpenClaw

### Pull Request Guidelines

- Keep descriptions concise but informative
- Verify links are working before submitting
- Follow the existing table format
- Add appropriate categories if needed
