# OpenClaw Resources

A curated collection of resources, tutorials, plugins, and tools for [OpenClaw](https://openclaw.ai) — the open-source AI agent framework that connects leading AI models to messaging apps you already use.

## What is OpenClaw?

OpenClaw is a free, open-source AI agent framework that connects AI models (ChatGPT, Claude, Gemini, DeepSeek) to messaging apps like Telegram, Discord, WhatsApp, Slack, Signal, and iMessage. It transforms them into always-on personal assistants that can actually do things for you.
